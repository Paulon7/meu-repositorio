Ficha Personagem em C++

Este projeto implementa um sistema de ficha de personagem para jogos de RPG, permitindo aos jogadores criar e gerenciar seus personagens de forma interativa. Utilizando C++, a aplicação fornece uma interface de linha de comando onde os usuários podem inserir detalhes sobre seus personagens, como nome, classe, nível, habilidades, e equipamentos.

Projeto Fazenda Inteligente

O projeto visa a automação de uma maquete de fazenda, através do desenvolvimento e implementação de um sistema inteligente de controle de iluminação. Utilizando a tecnologia Wi-Fi integrada a uma placa ESP32, o sistema permitirá o acionamento e desligamento remoto das luzes em diferentes áreas da maquete, como pasto e casa, por meio de um software acessível via site. Este projeto tem como objetivo melhorar a eficiência energética e a organização da maquete, proporcionando uma gestão elétrica mais prática e inovadora.
